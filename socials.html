Aggregator Tools

All-in-one widget options

Third-party tools that aggregate multiple social feeds into a single embeddable widget. Research findings below.

Elfsight

Most popular all-in-one. Individual widgets for Facebook, Instagram, Google Reviews, Yelp, TikTok, and LinkedIn. Each is a separate widget with its own embed code.

Free tier: 1 widget, 200 views/month
Basic: $5/mo per widget (unlimited views)
Embed: <script src="https://static.elfsight.com/platform/platform.js"></script>
+ <div class="elfsight-app-XXXXX"></div>
Free tier available

Curator.io

Social media aggregator that pulls feeds from Instagram, Facebook, TikTok, Twitter/X, LinkedIn, YouTube, and more into a single unified feed wall.

Free tier: 1 feed, 1 source, Curator branding
Pro: $25/mo (multiple sources, no branding)
Embed: JS snippet + <div id="curator-feed-XXXXX">
Layouts: Grid, Carousel, Waterfall, List
Free tier available

Trustindex.io

Specialized for review aggregation. Pulls Google, Yelp, Facebook reviews into beautiful badge/carousel widgets. Great for social proof.

Free tier: 1 widget, basic styling
Premium: $10/mo (multiple sources)
Best for: Review aggregation specifically
Embed: Single script tag
Free tier available

Juicer.io

Aggregates 15+ social networks into a single feed. Moderate/filter posts. Good for displaying a unified social wall on landing pages.

Free tier: 2 sources, Juicer branding
Small: $19/mo (5 sources, no branding)
Embed: <script src="https://assets.juicer.io/embed.js">
Free + Paid

EmbedSocial

Pull Instagram, Facebook, Google, TikTok feeds. Also generates review widgets. Strong moderation and analytics dashboard.

Free tier: Limited (1 source, branding)
PRO: $29/mo
Unique: Auto-generates review request links
Mostly paid

SociableKIT

Widget builder for Facebook feeds, Google Reviews, Yelp reviews, LinkedIn profiles. Each platform gets its own widget type.

Free tier: 1 widget with branding
Basic: $9/mo
Notable: Has Yelp-specific widget (rare)
Free tier available

Developer Reference

Embed methods per platform

Platform Embed Reference Table

Platform Official Method Static HTML? Auth Required? Best Free Option
Facebook Page Plugin iframe Yes - direct iframe No (public pages) Facebook Page Plugin (free, unlimited)
Yelp Badge image + link (official widget deprecated) Yes - badge image or manual No for badge; API key for Fusion API Manual review cards + Yelp link
Google Reviews Places API (JS library) No - requires API key + CORS proxy Yes (API key) Elfsight or Trustindex free widget
Instagram oEmbed via Graph API No - requires access token Yes (Facebook App + IG Business) Elfsight IG Feed widget (free tier)
TikTok oEmbed API (per-video only) Partial - individual videos via blockquote No for oEmbed TikTok embed.js for individual videos
LinkedIn None (deprecated 2018) No official embed N/A Manual profile card + link

Recommended Setup for Cesar & Associates

Phase 1 (Now): Use the Facebook Page Plugin iframe (already working above). Create manual Yelp/Google review cards with direct links. Deploy LinkedIn profile card. Keep IG and TikTok placeholders.

Phase 2 (When IG/TikTok launch): Sign up for Elfsight (free tier gets 1 widget). Use it for Instagram feed since IG has no free static embed. Embed individual TikTok videos using their free oEmbed/blockquote method.

Phase 3 (Scale): Consider Curator.io Pro ($25/mo) to aggregate all platforms into one unified social wall. Or use Elfsight Basic ($5/widget/mo) for individual high-quality widgets per platform.

Google Reviews: Get the Place ID from Google Business Profile settings. Then either use Elfsight's Google Reviews widget (free tier) or Trustindex.io for a review badge. The "Leave a Review" link format is: https://search.google.com/local/writereview?placeid=YOUR_PLACE_ID
